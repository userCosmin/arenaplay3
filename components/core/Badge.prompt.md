Uppercase mono chip for categories, tags and status — e.g. game category, "Multiplayer", age rating, "Disponibil".

```jsx
<Badge tone="primary">VR Dinamic</Badge>
<Badge tone="accent" solid>Disponibil</Badge>
<Badge tone="neutral" size="sm">10+</Badge>
```

Tones: `primary | secondary | accent | neutral | warning`. `solid` fills it; `size="sm"` for inline use.
