Opens every page section — uppercase mono eyebrow, display title, optional lead paragraph.

```jsx
<SectionHeading eyebrow="Experiențe imersive" title="Jocurile noastre"
  lead="De la free-roam la simulatoare — ceva pentru fiecare." align="center" />
```

`align="center"` for hero-style sections; `gradient` paints the title cyan→violet.
